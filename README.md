## What is SPFx Foundry Agents?

**Plug-and-play Azure AI Foundry agents in SharePoint and Teams.**

One SPFx solution runs on **SharePoint pages**, **Teams tabs**, and **Teams mobile**. No separate portal — agents live where people already work.

- **The problem:** Copilot is in Microsoft 365, but there’s no straightforward way to embed custom Azure AI Foundry agents (with tools, orchestration, governance) into SPFx across SharePoint and Teams.
- **The solution:** A reusable SPFx framework that hosts Foundry agents with secure authentication (no API keys, native Microsoft 365 sign-in, token for the right audience), context passing, and one UI across all surfaces.
- **Deploy once** — connect any Foundry agent endpoint, configure per agent (name, icon, prompts, audience, attachments), and reuse across sites and departments. Supports both delegated and application permissions.

**Categories:** Best use of SharePoint in AI agents · Most innovative SPFx experience · Best mobile experience.
